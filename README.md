# Movie Reviews - Vue Frontend

> Vue 3 frontend application for our Interactive Media Design movie review website project.

## Project Information

**Assignment:** Headless CMS Review Site  
**Course:** Interactive Media Design - Algonquin College  
**Due Date:** December 11, 2025

## Technologies Used

- **Framework:** Vue 3 (Composition API)
- **Build Tool:** Vite
- **Routing:** Vue Router
- **Styling:** Bootstrap 5
- **HTTP Client:** Fetch API

## Features

- Display list of movie reviews
- Search functionality
- Individual review pages
- Responsive design
- Site-wide navigation

## Setup Instructions

```bash
# Clone the repository
git clone https://github.com/tama0046/movie-reviews-frontend.git

# Navigate to project directory
cd movie-reviews-frontend

# Install dependencies
npm install

# Start development server
npm run dev
```
